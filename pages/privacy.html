Privacy Policy

1. Overview

This Privacy Policy describes how the “Join” project handles information in the context of a development and portfolio application created as part of a group training program. The project is not intended for public commercial use or deployment as a live production service. Its primary purpose is educational and demonstrative, showcasing development skills as part of a professional portfolio.

2. Scope of Data Processing

The Join project does not actively collect, process, or store personal data from external users in a commercial sense. Any data used within the application is either synthetic, test-related, or manually entered by users for demonstration purposes during development and testing phases.

3. Hosting and Accessibility

The application may be hosted in a limited or non-public environment (e.g. development servers, portfolio hosting, or educational platforms). Access is typically restricted to project contributors, instructors, or reviewers. No guarantee is made regarding public availability or long-term hosting stability.

4. Use of Cookies and Analytics

This project does not intentionally implement tracking technologies, advertising systems, or third-party analytics tools. If any browser storage (such as localStorage or sessionStorage) is used, it is strictly for functional purposes such as maintaining application state during usage.

5. Data Security

As this is a non-production educational project, standard production-grade security guarantees are not provided. However, care has been taken to follow best practices in coding and data handling within the scope of the training environment.

6. Contact

For questions regarding this project or its implementation, please contact the development team via the associated training or portfolio communication channels. This project is not operated as an official business entity.
